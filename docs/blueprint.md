# **App Name**: JH Smart Solutions Hub

## Core Features:

- Sticky Navigation Bar: Fixed top navigation bar with logo text "JH Smart Solutions" and smooth scrolling to different sections.
- Contact Form Submission: Store contact form submissions (Name, Phone, Email, Message) into Firestore collection named 'contacts'.
- Service Quote Request: Capture service quote requests from popup forms and store them in the 'leads' Firestore collection.
- Testimonial Display: Dynamically load and display client testimonials from Firestore collection 'testimonials' in a carousel or slider.
- Google Maps Integration: Embed a Google Maps component showing the shop location.
- Dynamic Content and SEO Tool: Use an AI tool to insert keywords into your website's description field for SEO improvement, leveraging provided data for reasoning.

## Style Guidelines:

- Primary color: Strong violet (#7950F2) to reinforce the modern, professional tone.
- Background color: Near-white, very light violet (#F4F2FC) to provide a clean, uncluttered backdrop.
- Accent color: Deep blue (#5079F2) for interactive elements, providing strong contrast against the violet and background.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use modern, flat-style icons to represent different services and sections.
- Fully responsive layout optimized for mobile and desktop devices using a grid-based system.
- Subtle hover animations on buttons and cards, with smooth scrolling transitions between sections.
